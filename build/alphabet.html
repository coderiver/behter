Alphabet

These illustrations were created as a result of a typographic project. It is a personal work. It was very useful experience which stimulated imagination and kept my brains and mind fresh over the last 30 days. One day, one letter, one story.

The result is a geometric alphabet poster. Each letter has it’s own story, it’s own meaning. Illustrations can be used separately, for different aspects of design.
